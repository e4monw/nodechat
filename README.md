# nodechat
Simple, lightweight node.js chat server and client
Code is based off the example at the socket.io website
Sounds by theta4 from freesound.org

## How to use
Preparation:
```
$ git clone https://github.com/fatbu/nodechat.git
$ cd nodechat
$ npm install
```

######Arguments:
`node app [port]`

```
$ node app 3000
listening on: 3000
address: ***.***.*.***
```
