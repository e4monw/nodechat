# nodechat
Simple, lightweight node.js chat server and client

Sounds by theta4 from freesound.org

## How to use
Preparation:
```
$ git clone https://github.com/fatbu/nodechat.git
$ cd nodechat
$ npm install
```

Arguments:
`node app [port]`

```
$ node app 3000
listening on: 3000
address: ***.***.*.***
```
